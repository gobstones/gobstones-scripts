(static) copyFilesFrom(fromFolder, toFolder, overwrite, toHide) → {Array.<string>}

Copy all files from a folder to another. If overwrite is true, any existing file will be overwritten. The toHide array may contain a set of files or folder names that will be prefixed with "." after copying.

Parameters:
Name Type Description

The folder to copy from.
toFolder string

The folder to copy to.

Wether or not to overwrite the files already present.

A list of files to hide after copying.

(static) getPackageManager() → {string}

Return the package manager in use based different features. First, by identifying the current runner through the "npm_config_user_agent" environment variable. It such variable is not set, which is common for global runs, attempts to identify the runner by locating the global "gobstones-scripts" command. If no match is found, defaults to npm.

Returns:
Type
string
