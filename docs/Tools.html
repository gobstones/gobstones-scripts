Tools

Author:
  • Alan Rodas Bonjour

This module provides tooling to be used on the package-scripts.js file, so it helps on writing nps commands. Every function exported handles automatically the configuration file to be used when run from inside gobstones-scripts

Methods

(static) chmod(options) → {string}

Perform a chmod on a file or directory, setting specific permissions on it.

Example
chmod({  files: './dist/files', mod: '+x' })
Parameters:
Name Type Description
options object

Options for "chmod" command:

Properties
Name Type Description
files string

The files or folder to apply permissions to, may be a glob pattern
mod string

The permissions to apply
Returns:
Type
string

(static) concurrent(…tasks) → {string}

Run a set of actions concurrently. Pass an object as the sole argument if you want to associate names to each process, or multiple arguments if the names of the actions are not relevant in output.

Examples
concurrent(nps('build'), serve('./dist'))
concurrent({
    build: nps('build'),
    serve: serve('./dist')
})
Parameters:
Name Type Attributes Description
tasks Object.<string, string> | Array.<string> <repeatable>

"concurrent" expects an object with keys as the task name, and the actual task as value. Or, if you do not care about task names being reported, multiple arguments can be passed.
Returns:
Type
string
